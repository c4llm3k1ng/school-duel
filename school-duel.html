🔥 Firebase einrichten

Für den Online-Duell-Modus benötigst du ein kostenloses Firebase-Projekt mit Realtime Database.

Einmalig einrichten (ca. 3 Min.):
1. console.firebase.google.com → Projekt erstellen
2. Build → Realtime Database → Erstellen (Region: europe-west1)
3. Regeln-Tab → ".read": true, ".write": true → Veröffentlichen
4. Projekteinstellungen → Allgemein → API-Key & Projekt-ID kopieren
